## 🚀 Instalación y Despliegue

Este proyecto está diseñado para funcionar directamente desde el navegador y es totalmente compatible con **GitHub Pages**.

1. Sube los archivos `App.jsx`, `index.html` y `manifest.json` a tu repositorio.
2. En GitHub, ve a **Settings > Pages**.
3. Selecciona la rama `main` y guarda.
4. Tu panel estará disponible en: `https://tu-usuario.github.io/tu-nombre-de-repo/`

## 🎥 Configuración en OBS Studio

Para integrar los gráficos en tu transmisión, sigue estos pasos:

1. **Abrir el Panel:** Abre la URL de tu proyecto en Chrome o Edge.
2. **Copiar Link de Overlay:** Haz clic en el botón **"COPIAR URL OBS"** en el menú lateral.
3. **En OBS:**
   - Añade una nueva fuente de tipo **Navegador (Browser)**.
   - Pega el link copiado (asegúrate de que termine en `?mode=overlay`).
   - Configura el ancho en `1920` y el alto en `1080`.
   - Marca la casilla **"Actualizar el navegador cuando la fuente se active"**.
   - El sistema detectará automáticamente el modo overlay y mostrará una salida 100% transparente.
